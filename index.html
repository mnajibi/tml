Welcome to tml - The Text to HTML Converter!

Overview

tml is a command-line tool that efficiently converts `.txt` files into `.html` format. It automatically wraps paragraphs and offers a range of customization features.

How to Use

            
./main.py <path-to-txt-file>

Features

  • Converts single or multiple `.txt` files to `.html`.
  • Automatic paragraph wrapping.
  • Specify custom output directories.

For detailed documentation, source code, and contributions, visit the GitHub Repository.
